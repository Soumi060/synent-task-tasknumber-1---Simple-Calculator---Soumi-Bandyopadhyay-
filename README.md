Simple Calculator (CLI)

A beginner-friendly Command Line Interface (CLI) calculator built using Python.

Features

* Addition
* Subtraction
* Multiplication
* Division
* User input support
* Error handling for invalid inputs
* Prevents division by zero

---

Technologies Used

* Python 3

---

Project Structure

```text
simple-calculator/
│── calculator.py
│── README.md
```

---
How to Run

1. Clone the Repository

```bash
git clone https://github.com/your-username/simple-calculator.git
```

2. Navigate to the Project Folder

```bash
cd simple-calculator
```

3. Run the Program

```bash
python calculator.py
```

Error Handling

The program handles:

* Invalid menu choices
* Non-numeric inputs
* Division by zero
